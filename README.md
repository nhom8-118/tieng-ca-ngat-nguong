# Tiếng ca ngất ngưởng giữa dòng thế sự

> "Vũ trụ nội mạc phi phận sự..." - Một dự án thuyết trình Văn học 11 về cụ Nguyễn Công Trứ.

## 🛠 Tech Stack
- **Framework:** React + Vite (Vì cụ Trứ mà sống ở 2026 chắc chắn cũng dùng Vite)
- **Styling:** Tailwind CSS
- **Deployment:** GitHub Actions (Robot tự động nấu code)
- **Host:** GitHub Pages

## 🖋 Ghi chú cho những kẻ tò mò
Nếu bạn đang đọc dòng này, chúc mừng, bạn đã mò vào tận "hậu trường" của nhóm 8 11/9. 
Web này được dựng lên không chỉ để lấy điểm Văn, mà còn để khẳng định rằng: Dân khối tự nhiên/kỹ thuật một khi đã làm Văn thì nó phải ở cái tầm... "ngất ngưởng" (nếu sài AI).
